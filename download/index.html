NoCloud Desktop

Unsigned builds — no Apple/Microsoft certificate yet, so your OS will warn on first open. Safe to proceed: NoCloud only talks to your own server.
  • macOS: right-click the app → Open, or check System Settings for an "Open Anyway" prompt under Gatekeeper/security (exact wording varies by macOS version)
  • Windows: "More info → Run anyway" on the SmartScreen prompt. The installer downloads the Microsoft Edge WebView2 runtime automatically if your PC doesn't have it. Note: on Windows the app currently offers the built-in demo and connecting to an existing portal; setting up a new local instance needs WSL (or a Mac/Linux machine)
  • Linux: mark the AppImage executable, or install the .deb

This is the desktop wrapper around your own self-hosted instance — it opens a native window on the web UI served by your Raspberry Pi (or any Linux machine running NoCloud). Your data never leaves your machine. See Getting started to set up the server first.

Verify your download against SHA256SUMS.txt — with unsigned installers, the checksum is your integrity guarantee. Installers are built locally from each tagged desktop release.
