Download NoCloud

Native desktop wrapper around your self-hosted instance.

These builds are unsigned. No Apple notarization or Microsoft Authenticode certificate exists for this project yet. macOS will show a Gatekeeper warning — right-click the .dmg and choose Open to bypass it. Windows SmartScreen will flag the installer as unrecognized — click "More info → Run anyway." Linux users can mark the AppImage executable and run it directly. The app connects only to your own server; it contains no data, credentials, or cloud tokens.

This is the desktop wrapper around your own self-hosted instance — it opens a native window on the web UI served by your Raspberry Pi (or any Linux machine running NoCloud). Your data never leaves your machine. See Getting started to set up the server first.

Source on GitHub. Installers are built from the latest main by this CI workflow.
